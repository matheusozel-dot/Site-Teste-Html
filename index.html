Anotações curso de HTML

1º - Formatação de Texto

TAGS

AS tags são basicamente a alma da programção em HMTL Tudo a qual será programado, será preciso ter
"<" e ">" para abrir e fechar a linha de codigo.

E para Códigos especificos, será nescessario na hora de fechar a tag uma barra "/" para indicar o fechamento.


Tags Principais

"< !doctype html >" - Indica a versão do HTML que esta sendo ultilizada, nesse caso a versão 5.0 .

"< Head >" - O que somente o Navegador irá ler e compreender para executar.

"< Body >" - Corpo do site, onde tudo o que for programado irá aparecer.

"< tittle >" - Literalmente o titulo da pagina, formata o texto para algo maior.

"< h1 >" - formata o tamnho do texto, indo de 1 a 6.

"< p >" - Formata o texto para paragrafo, ou seja, um texto corrido.

"< b >" - Formata o texto para negrito

"< strong >" - Igual ao "< b >" mas com maior ênfase.

"< hr >" - adicionar uma linha horizontal para separa sessões de outras.

"< br >" - quebra de linha.

"< i >" - formatar texto em italico.

"< u >" - formatar texto em sublinhado.

"< small >" - formatar texto para menor tamnho

"< del >" - riscar texto no meio.

"< sup >" - formatar texto na parte superior do texto.

"< sub >" - formatar texto na parte inferior do texto.

"< mark >" - marcação
